# 外部設計書 インデックス

## プロジェクト概要

日本語オンライン書店アプリ（購買フロー特化版）の外部設計書集。要件定義書（`user_requirements.md`）に記載された各機能の外部仕様を機能単位で文書化したもの。

### 技術スタック

| サービス | スタック | ポート |
|---|---|---|
| frontend | Next.js 14 + TypeScript | 3000 |
| backend | Express.js + TypeScript | 4000 |
| mysql | MySQL 8 | 3306 |

### 画面遷移フロー

```
商品一覧 (/) → 商品詳細 (/products/[id]) → カート (/cart) → 注文フォーム (/order) → 注文完了 (/order/complete)
```

## 機能概要

### F-01: 商品一覧表示機能
販売中の全書籍をグリッド形式で表示するトップページ。書影・タイトル・著者・価格を表示し、各カードから商品詳細画面へ遷移できる。`GET /products` を使用。

### F-02: 商品詳細表示・カート追加機能
個別書籍の詳細情報（書影・タイトル・著者・価格・説明文）を表示し、「カートに追加」ボタンでカートへ追加できる画面。`GET /products/:id` を使用。カート操作は `CartContext` 経由で `localStorage` に永続化。

### F-03: カート管理機能
カート内アイテムの確認・数量変更・削除・合計金額表示を行う画面。クライアントサイド完結（API呼び出しなし）。`CartContext` の `items, updateQuantity, removeItem, total` を使用。

### F-04: 注文フォーム機能
配送先情報（氏名・住所・メールアドレス）の入力と注文サマリー表示を行う2カラムレイアウト画面。バリデーション後、`POST /orders` を実行して注文確定。成功時は注文完了画面へリダイレクト。

### F-05: 注文完了表示機能
注文完了メッセージと注文番号を表示するシンプルな完了画面。クエリパラメータ `orderNumber` から注文番号を取得。商品一覧へ戻るリンクを提供。

## 機能間の依存関係

```
[F-01 商品一覧]
     ↓ カードクリック
[F-02 商品詳細] ──────┐
     ↓ カートに追加     │
     │                │ ヘッダーのカートリンク
     ↓                │
[F-03 カート] ←───────┘
     ↓ 注文手続き
[F-04 注文フォーム]
     ↓ POST /orders 成功
[F-05 注文完了]
     ↓ 商品一覧に戻る
[F-01 商品一覧]
```

## 共通仕様

### スコープ外機能（要件書 3章）

- ログイン・会員管理
- 決済処理
- 在庫管理
- 管理画面
- レビュー・評価
- 検索・フィルター

### スタイル規約

- スタイルはすべてインラインスタイル（`style={{...}}`）で記述
- CSS モジュール・Tailwind・外部 CSS フレームワークは不使用
- カラーパレット:
  - メインネイビー: `#1e3a8a` / `#1e40af`
  - ページ背景グレー: `#f3f4f6`
  - カード背景: `white` + `box-shadow: 0 2px 8px rgba(0,0,0,0.08)`
  - テキスト（本文）: `#111827` / `#374151`
  - テキスト（サブ）: `#6b7280`
  - エラー・削除: `#ef4444`

### コンポーネント規約

- すべてのページコンポーネントは `"use client"` ディレクティブを先頭に記述

### 状態管理

- `CartContext`（`src/contexts/CartContext.tsx`）がアプリ全体をラップ（`Providers.tsx` 経由）
- カートは `localStorage` に永続化
- 公開メンバー: `items, addItem, updateQuantity, removeItem, clearCart, total`

### API通信

- ベースURL: `process.env.NEXT_PUBLIC_API_URL`（デフォルト `http://localhost:4000`）
- 認証なし（公開エンドポイント）

### バックエンドAPI一覧

| メソッド | パス | 用途 | 使用機能 |
|---|---|---|---|
| GET | `/health` | ヘルスチェック | なし（運用用途） |
| GET | `/products` | 書籍一覧取得 | F-01 |
| GET | `/products/:id` | 書籍詳細取得 | F-02 |
| POST | `/orders` | 注文作成 | F-04 |

### データベーススキーマ

- **books**: `id, title, author, price (INT 円), description, image_url, created_at`
- **orders**: `id, order_number (UNIQUE), name, address, email, total_amount (INT 円), created_at`
- **order_items**: `id, order_id (FK→orders), book_id, title, price, quantity`

## 要件トレーサビリティマトリクス

| 要件ID | 要件概要 | 担当機能 |
|---|---|---|

| U-05 | カートに追加したい | F-02 |

| U-07 | カート内の本一覧を確認したい | F-03 |
| U-08 | 冊数を変更したい（リアルタイム反映） | F-03 |
| U-09 | カートから削除したい | F-03 |
| U-10 | 合計金額を確認したい | F-03 |
| U-11 | 注文手続きに進みたい | F-03 → F-04 |
| U-12 | 氏名・住所・メールを入力したい | F-04 |
| U-13 | 入力ミスに気づきたい | F-04 |
| U-14 | 注文内容を確認しながら入力したい | F-04 |
| U-15 | 入力完了後に注文を確定したい | F-04 → F-05 |
| U-16 | 注文受付の確認をしたい | F-05 |
| U-17 | 注文番号を受け取りたい | F-05 |
| U-18 | 注文後にトップへ戻りたい | F-05 → F-01 |
