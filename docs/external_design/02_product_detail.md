# 外部設計書（機能仕様書）
## 画面2：商品詳細

**対応要件:** U-04, U-05, U-06
**画面パス:** `/books/[id]`
**Nextコンポーネント:** `src/app/books/[id]/page.tsx`（サーバーコンポーネント）

---

## 1. 画面概要

選択した書籍の詳細情報を表示し、カートへの追加を行う画面。書影・タイトル・著者・価格・説明文を大きく表示する。「カートに追加」後もこの画面にとどまり、他の書籍を引き続き閲覧できる。

---

## 2. 画面レイアウト（ワイヤーフレーム）

```
┌─────────────────────────────────────────────────────┐
│ [ロゴ / サイト名]                   [カート(n冊)] │  ← Header（共通）
├─────────────────────────────────────────────────────┤
│                                                     │
│  ← 商品一覧へ戻る                                  │
│                                                     │
│  ┌──────────────────┐  ┌────────────────────────┐  │
│  │                  │  │ タイトル               │  │
│  │    [書影画像]    │  │ 著者：著者名           │  │
│  │                  │  │ 価格：¥x,xxx           │  │
│  │                  │  │                        │  │
│  │                  │  │ ─────────────────────  │  │
│  │                  │  │ 説明文                 │  │
│  │                  │  │ （複数行テキスト）     │  │
│  └──────────────────┘  │                        │  │
│                         │  ┌──────────────────┐  │  │
│                         │  │  カートに追加     │  │  │
│                         │  └──────────────────┘  │  │
│                         └────────────────────────┘  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**レイアウト仕様:**
- PC（≥768px）：書影と詳細情報を横並び（左：書影、右：テキスト+ボタン）
- モバイル（<768px）：書影を上、詳細情報を下に縦積み
- 書影最大幅：320px（レスポンシブで縮小）

## 3. 表示項目と表示条件

### 3-1. ヘッダー（共通）

| 要素 | 表示内容 | 表示条件 |
|---|---|---|
| ロゴ/サイト名 | 書店名テキスト（リンク：`/`） | 常時表示 |
| カートアイコン | カートに入っている総冊数 | 常時表示 |

### 3-2. パンくず/戻るリンク

| 要素 | 表示内容 | 表示条件 |
|---|---|---|
| 戻るリンク | 「← 商品一覧へ戻る」（リンク：`/`） | 常時表示 |

### 3-3. 書籍詳細情報

| 要素 | 表示内容 | 表示条件 |
|---|---|---|
| 書影 | `Book.coverImageUrl` の画像 | 常時表示。ロード失敗時はプレースホルダー表示 |
| タイトル | `Book.title` | 常時表示 |
| 著者 | `著者：{Book.author}` | 常時表示 |
| 価格 | `¥{Book.price.toLocaleString()}（税込）` | 常時表示 |
| 説明文 | `Book.description` | 常時表示（改行を保持） |

### 3-4. カートに追加ボタン

| 要素 | 表示内容 | 表示条件 |
|---|---|---|
| ボタン | 「カートに追加」 | 常時表示 |
| 追加後フィードバック | ボタンラベルを一時的に「追加しました ✓」に変化させ、2秒後に元に戻す | カートへの追加操作直後 |

### 3-5. エラー状態

| 条件 | 表示内容 |
|---|---|
| 指定 `id` の書籍が存在しない（API 404） | `not-found.tsx` に委譲し「書籍が見つかりません」を表示 |
| APIエラー（500等） | 「書籍情報の取得に失敗しました。しばらくしてから再度お試しください。」を表示 |

## 4. ユーザー操作と動作

| 操作対象 | 操作 | 動作 |
|---|---|---|
| 「← 商品一覧へ戻る」リンク | クリック | 商品一覧画面（`/`）へ遷移 |
| 「カートに追加」ボタン | クリック | `CartContext` の `add(bookId)` を呼び出す。`CartItem` が既存の場合は `quantity` を +1 する。ボタンラベルを「追加しました ✓」に変えて2秒後に元に戻す（画面遷移しない） |
| カートアイコン（ヘッダー） | クリック | カート画面（`/cart`）へ遷移 |
| ロゴ/サイト名（ヘッダー） | クリック | 商品一覧画面（`/`）へ遷移 |

**カート追加の詳細ロジック:**

```
if (cartItems に bookId が存在する) {
  既存 CartItem の quantity を +1
} else {
  新規 CartItem { bookId, quantity: 1 } を追加
}
localStorage に最新の cartItems を保存
```

---

## 5. バリデーションルール

なし（カート追加は数量を固定で +1 するため入力なし）

---

## 6. 使用するAPIエンドポイント

| タイミング | メソッド | パス | 説明 |
|---|---|---|---|
| ページ初期表示（SSR） | `GET` | `/api/books/:id` | 指定IDの書籍詳細を取得 |

**データ取得方式:** Next.js サーバーコンポーネントの `fetch` で SSR 取得。存在しないIDの場合は `notFound()` を呼び出す。

---

## 7. 画面遷移先

| 遷移先 | 条件 |
|---|---|
| 商品一覧画面 `/` | 「← 商品一覧へ戻る」リンクをクリック |
| カート画面 `/cart` | ヘッダーのカートアイコンをクリック |
| （画面遷移なし） | 「カートに追加」ボタンをクリック（現在画面に留まる） |
