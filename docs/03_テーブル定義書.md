## 1. books（書籍）

### テーブル定義

| カラム名 | 論理名 | データ型 | PK | NOT NULL | デフォルト | 説明 |
|---|---|---|---|---|---|---|
| id | 書籍ID | INT | ○ | ○ | AUTO_INCREMENT | サロゲートキー |
| title | タイトル | VARCHAR(255) | - | ○ | - | 書籍タイトル |
| author | 著者 | VARCHAR(255) | - | ○ | - | 著者名 |
| price | 価格 | INT | - | ○ | - | 税込価格（円）。CHECK >= 0 |
| description | 説明文 | TEXT | - | - | NULL | 書籍の紹介文 |
| image_url | 書影URL | VARCHAR(500) | - | - | NULL | 書影画像の URL |
| created_at | 作成日時 | TIMESTAMP | - | ○ | CURRENT_TIMESTAMP | レコード作成日時 |
| updated_at | 更新日時 | TIMESTAMP | - | ○ | CURRENT_TIMESTAMP ON UPDATE | レコード更新日時 |

### DDL

```sql
CREATE TABLE IF NOT EXISTS books (
  id          INT AUTO_INCREMENT PRIMARY KEY,
  title       VARCHAR(255)   NOT NULL,
  author      VARCHAR(255)   NOT NULL,
  price       INT            NOT NULL CHECK (price >= 0),
  description TEXT,
  image_url   VARCHAR(500),
  created_at  TIMESTAMP      DEFAULT CURRENT_TIMESTAMP,
  updated_at  TIMESTAMP      DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 2. orders（注文）

### テーブル定義

| カラム名 | 論理名 | データ型 | PK | NOT NULL | デフォルト | 説明 |
|---|---|---|---|---|---|---|
| id | 注文ID | INT | ○ | ○ | AUTO_INCREMENT | サロゲートキー |
| customer_name | 顧客氏名 | VARCHAR(100) | - | ○ | - | 注文者の氏名 |
| customer_address | 顧客住所 | VARCHAR(500) | - | ○ | - | お届け先住所 |
| customer_email | 顧客メール | VARCHAR(255) | - | ○ | - | 注文者のメールアドレス |
| total_amount | 合計金額 | INT | - | ○ | - | 注文合計金額（円）。CHECK >= 0 |
| created_at | 注文日時 | TIMESTAMP | - | ○ | CURRENT_TIMESTAMP | 注文が作成された日時 |

### DDL

```sql
CREATE TABLE IF NOT EXISTS orders (
  id               INT AUTO_INCREMENT PRIMARY KEY,
  customer_name    VARCHAR(100)   NOT NULL,
  customer_address VARCHAR(500)   NOT NULL,
  customer_email   VARCHAR(255)   NOT NULL,
  total_amount     INT            NOT NULL CHECK (total_amount >= 0),
  created_at       TIMESTAMP      DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

### 合計金額の算出

`total_amount = Σ (unit_price × quantity)` for all items

## 3. order_items（注文明細）

### テーブル定義

| カラム名 | 論理名 | データ型 | PK | NOT NULL | デフォルト | 説明 |
|---|---|---|---|---|---|---|
| id | 明細ID | INT | ○ | ○ | AUTO_INCREMENT | サロゲートキー |
| order_id | 注文ID | INT | - | ○ | - | orders.id への外部キー |
| book_id | 書籍ID | INT | - | ○ | - | books.id への外部キー |
| quantity | 数量 | INT | - | ○ | - | 注文数量。CHECK > 0 |
| unit_price | 単価 | INT | - | ○ | - | 注文時点の単価（価格変動への対応）。CHECK >= 0 |

### DDL

```sql
CREATE TABLE IF NOT EXISTS order_items (
  id         INT AUTO_INCREMENT PRIMARY KEY,
  order_id   INT  NOT NULL,
  book_id    INT  NOT NULL,
  quantity   INT  NOT NULL CHECK (quantity > 0),
  unit_price INT  NOT NULL CHECK (unit_price >= 0),
  FOREIGN KEY (order_id) REFERENCES orders(id) ON DELETE CASCADE,
  FOREIGN KEY (book_id)  REFERENCES books(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

### 制約

| 制約名 | 種別 | カラム | 参照先 | 削除時動作 |
|---|---|---|---|---|
| FK order_items → orders | FOREIGN KEY | order_id | orders(id) | CASCADE（注文削除で明細も削除） |
| FK order_items → books | FOREIGN KEY | book_id | books(id) | RESTRICT |

| orders | customer_email | メールアドレスによる注文検索 |
